3. Pre-Event Checklist

            
Day Before

            
    
                
  • All Docker containers on KoTH server running (docker ps)
    • 
                
  • All Hills accessible and SLA OK
    • 
                
  • VPN server running, WireGuard configs generated
    • 
                
  • Scorebot healthcheck all hills: curl http://YOUR_SERVER:8080/api/v1/batch-check
    • 
                
  • Participant registration ready / registration code set
    • 
                
  • Test start → stop game, verify scoring works
    • 
                
  • Backup database: docker exec koth-postgres pg_dump -U koth_admin koth_ctf > backup.sql
    • 
            

            
Verify Hills

            
curl -s http://YOUR_SERVER:8080/api/v1/batch-check | python3 -m json.tool

            
Verify Agent Status

            
curl -s http://YOUR_SERVER:8000/api/agent/status | python3 -m json.tool


            
            
4. Day-of-Event Flow

            
Adapt this timeline to your event schedule:

            

                
TimeAction

                
                    
T-2hVerify all servers UP — docker ps

                    
T-1.5hEnable registration

                    
T-1hGate open — participants register

                    
T-30mTechnical briefing

                    
T-5mDisable registration

                    
T-2mBroadcast: "Game starts in 2 minutes!"

                    
T=0🚀 START GAME

                    
T+0-5mGrace period 5 min (no scoring)

                    
T+5mTick #1 begins

                    
T-30m endScoreboard auto-freeze

                    
T=end🏁 GAME OVER

                    
T+5m endUnfreeze scoreboard

                    
T+10m endExport data & announce winner

                
            


            
            
5. Game Lifecycle Control

            
Start Game

            
curl -X POST http://YOUR_SERVER/api/admin/game/control \
  -H "Content-Type: application/json" \
  -H "X-Admin-Token: YOUR_ADMIN_TOKEN" \
  -d '{"action": "start"}'

            
After start, there is a 5-minute grace period before the first tick begins.

            
Pause / Resume / Stop

            
# Pause
curl -X POST .../api/admin/game/control -d '{"action": "pause"}'
# Resume
curl -X POST .../api/admin/game/control -d '{"action": "resume"}'
# Stop (PERMANENT!)
curl -X POST .../api/admin/game/control -d '{"action": "stop"}'

            
⚠️ Stop bersifat final. Game tidak bisa di-resume setelah stop.

            
Status Game

            

                
StatusKeterangan

                
                    
idleGame belum dimulai

                    
grace_periodDalam masa grace period (5 menit)

                    
runningGame berjalan, scoring aktif

                    
pausedGame di-pause

                    
stoppedGame selesai

                
            


            
            
6. Team Management

            
Create Team

            
curl -X POST http://YOUR_SERVER/api/teams \
  -H "Content-Type: application/json" \
  -H "X-Admin-Token: YOUR_ADMIN_TOKEN" \
  -d '{"name":"team-alpha","display_name":"Team Alpha","category":"default","vpn_ip":"10.10.0.10"}'

            
Bulk Create Teams

            
curl -X POST http://YOUR_SERVER/api/teams/bulk \
  -H "Content-Type: application/json" \
  -H "X-Admin-Token: YOUR_ADMIN_TOKEN" \
  -d '{"teams":[
    {"name":"team-alpha","display_name":"Team Alpha","category":"default"},
    {"name":"team-bravo","display_name":"Team Bravo","category":"default"}
  ]}'

            
Team Categories

            
Categories are configurable. The default category is default. You can define custom categories (e.g., military, civilian) to suit your event.


            
            
7. Registration Management

            
Toggle ON/OFF via Organizer Console or API:

            
curl -X PUT http://YOUR_SERVER/api/admin/registration/toggle \
  -H "X-Admin-Token: YOUR_ADMIN_TOKEN"

            
Registration code is set via REGISTRATION_CODE in .env

            
⚠️ Share the registration code during briefing, not before.

Toggle Hill Active/Inactive

            
curl -X PUT http://YOUR_SERVER/api/admin/hill/3/toggle \
  -H "X-Admin-Token: YOUR_ADMIN_TOKEN"

            
Useful if there are technical issues on a hill.

            
Regenerate Agent Token

            
curl -X POST http://YOUR_SERVER/api/admin/hills/3/agent-token \
  -H "X-Admin-Token: YOUR_ADMIN_TOKEN"


            
            
9. Scoreboard Control

            
Freeze / Unfreeze

            
# Freeze
curl -X POST http://YOUR_SERVER/api/admin/scoreboard/freeze \
  -H "X-Admin-Token: YOUR_ADMIN_TOKEN"

# Unfreeze
curl -X POST http://YOUR_SERVER/api/admin/scoreboard/unfreeze \
  -H "X-Admin-Token: YOUR_ADMIN_TOKEN"

            
Auto-freeze: Scoreboard automatically freezes 30 minutes before game end. After game stop, manually unfreeze to reveal final scores.


            
            
10. Score Adjustment

            
# Add 100 points to team ID 5
curl -X POST http://YOUR_SERVER/api/admin/score/adjust \
  -H "Content-Type: application/json" \
  -H "X-Admin-Token: YOUR_ADMIN_TOKEN" \
  -d '{"team_id":5,"points":100,"reason":"Bonus challenge"}'

# Deduct 50 points (penalty)
curl -X POST .../api/admin/score/adjust \
  -d '{"team_id":3,"points":-50,"reason":"Penalty: service destruction"}'

            
All adjustments are recorded in the audit log with the reason.

16. Troubleshooting

            
Hill SLA Failing

            
# Test from KoTH server (replace IPs with your hill VPC IPs)
curl -s -o /dev/null -w "%{http_code}" http://HILL1_VPC_IP:80
nc -zv HILL2_VPC_IP 9999
curl -s -o /dev/null -w "%{http_code}" http://PIVOT_VPC_IP:8080
nc -zv PIVOT_VPC_IP 27017

            
Restart Container on Hill

            
# Example: restart a hill container
ssh root@HILL1_PUBLIC_IP "docker restart container-name"
# Pivot server containers
ssh root@PIVOT_PUBLIC_IP "docker restart container-name"

            
Scorebot Error

            
docker logs --tail 50 koth-scorebot
docker restart koth-scorebot

            
Agent Not Reporting

            
# Check status
curl http://YOUR_SERVER/api/agent/status
# Restart agent on a hill
ssh root@HILL_PUBLIC_IP "systemctl restart koth-agent"

            
Game Reset (DESTRUCTIVE!)

            
⚠️ EXTREME CAUTION: Reset deletes ALL scoring data. Teams remain but all points reset to 0.

            
# Stop first!
curl -X POST .../api/admin/game/control -d '{"action":"stop"}'
# Reset
curl -X POST http://YOUR_SERVER/api/admin/game/reset \
  -H "X-Admin-Token: YOUR_ADMIN_TOKEN"

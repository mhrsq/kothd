📖 How to Connect — WireGuard

1

Install WireGuard

sudo apt update && sudo apt install -y wireguard

Windows: Download dari wireguard.com/install

2

Download & Import Config

Klik tombol "🛡️ Download WireGuard Config" di atas, lalu:

Linux:

sudo cp wg-team.conf /etc/wireguard/wg0.conf

Windows: Buka WireGuard GUI → Add Tunnel → pilih file wg-team.conf

3

Aktifkan Koneksi

sudo wg-quick up wg0

Windows: Klik Activate di WireGuard GUI

4

Verifikasi Koneksi

Cek status WireGuard:

sudo wg show

Scan VPN network untuk discovery:

nmap -sn 10.0.0.0/24

Scan port target yang ditemukan:

nmap -sT -Pn <target-ip>
5

Matikan VPN (jika perlu)

sudo wg-quick down wg0

📖 How to Connect — OpenVPN

1

Install OpenVPN

sudo apt update && sudo apt install -y openvpn

Windows: Download dari openvpn.net/community-downloads

2

Download & Connect

Klik tombol "🔑 Download OpenVPN Config" di atas, lalu:

Linux:

sudo openvpn --config team.ovpn

Tunggu sampai muncul: Initialization Sequence Completed

Windows: Buka OpenVPN GUI → Import file team.ovpn → Connect

3

Verifikasi Koneksi

Cek interface tunnel:

ip addr show tun0

Ping ke VPN network:

ping -c 3 <koth-server-ip>

⚠️ Troubleshooting

🔴 Tidak bisa ping?▶
Pastikan VPN aktif (sudo wg show), cek apakah firewall lokal memblokir, dan pastikan routing ke 10.0.0.0/24 melewati interface VPN.
🔴 DNS tidak resolve?▶
Gunakan IP langsung. Hills diakses via IP, bukan domain.
🔴 Koneksi lambat?▶
Cek MTU: sudo ip link set mtu 1280 dev wg0
🔴 Hill 3 & 4 unreachable?▶
Hills ini di belakang Pivot. Pastikan routing lewat pivot aktif. Anda mungkin perlu pivot melalui DMZ.

📋 Peraturan & Mekanisme Scoring

🎯 Tujuan Permainan

Kumpulkan poin sebanyak mungkin dengan menguasai (capture) dan mempertahankan (defend) server Hill. Setiap 60 detik, scorebot akan mengecek siapa "King" di setiap Hill. King mendapat poin!

⚡ Scoring Formula

👑
King Points = BASE (10) × Multiplier per tick per hill
🩸
First Blood = +50 bonus poin (pertama capture hill)
🛡️
Defense Streak = +5 bonus per tick berturut-turut memegang hill
🔄
Pivot Bonus = Hill di belakang pivot punya multiplier 1.5x
⚠️
SLA Required! Service hill HARUS tetap hidup. Jika service mati = 0 poin meskipun anda King!

📏 Aturan Pertandingan

✅
Exploit vulnerability yang ada di setiap Hill
✅
Gunakan tools apapun (Kali Linux, custom scripts, dll)
✅
Serang Hill milik team lain untuk merebut King
❌
DILARANG menyerang infrastruktur (VPN server, KoTH server, Scorebot)
❌
DILARANG melakukan DoS ke VPN/scoreboard
❌
DILARANG mematikan service Hill secara sengaja (penalty!)

👑 Cara Claim Hill

1

Dapatkan Root Access di Host OS Hill

Exploit vuln service → shell user → privilege escalation → root. Vuln services berjalan di Docker container, king.txt ada di host OS.

2

Tulis Nama Team ke /root/king.txt

echo "team-name" > /root/king.txt

⚠️ Nama harus PERSIS sama dengan team name yang terdaftar (case-sensitive). Cek di info team anda.

3

Pertahankan Akses

Scorebot baca king.txt setiap 60 detik. Jika team lain overwrite → mereka jadi King. Defend!
